Stack

A stack is an abstract data type that represents a collection of elements with a bounded capacity, operating on a Last In, First Out (LIFO) basis. It allows for the addition of elements (push) only at the top and removal of elements (pop) only from the top, similar to a pile of objects.

Basic features of a stack:

  • Ordered List: Contains similar data types.
  • LIFO Structure: Last In, First Out (most recent added is removed first).
  • Push/Pop Operations:
    • push(): Adds an element to the top.
    • pop(): Removes the top element.
  • Overflow/Underflow:
    • Overflow: Stack is full (cannot push).
    • Underflow: Stack is empty (cannot pop).

Applications:

  • Function call management
  • Expression evaluation
  • Undo mechanisms in applications
